Запуск репозитория:

Создание виртуального окружения

	python -m venv/virtualenv env

Желательно проверить pip на новые версии

	pip install --upgrade pip

Установка модулей

	pip install -r requirements.txt

Запуск сервера

	python manage.py runserver
